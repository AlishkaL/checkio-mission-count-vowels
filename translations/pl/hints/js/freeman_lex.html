Nie mam pojęcia, jak zacząć rozwiązywać tę misję.

Proszę złapać pomysł 💡! Proszę iterować po opuszczonym tekście i zwiększać predefiniowaną zmienną licznika za każdym razem, gdy znak jest samogłoską.

Ok, od czego powinienem zacząć?

Proszę zadeklarować zmienną count o wartości początkowej 0, którą zamierza Pan zwrócić na końcu. 

let count = 0;
...
return count;
Co robić dalej?

Proszę zmienić podany adres text na małe litery za pomocą metody string.toLowerCase()... 

text.toLowerCase()

... i iterować po nim za pomocą pętli for. 

for(let char of text.toLowerCase()) {
    ...
}
Nadal potrzebuję podpowiedzi.

Proszę sprawdzić, czy char jest samogłoską za pomocą metody string.includes(). 

"aeiou".includes(char)

Wyrażenie zwraca wartość logiczną, więc należy przekonwertować ją na liczbę całkowitą za pomocą funkcji Number(). Następnie proszę zwiększyć licznik count o tę wartość. 

count += Number("aeiou".includes(char));
Jak powinno wyglądać moje ostateczne rozwiązanie?

Wystarczy połączyć wszystkie poprzednie podpowiedzi i zawinąć kod w wyrażenie funkcyjne. Używanie podpowiedzi typu dla danych wejściowych i wyjściowych jest wysoce zalecane.

Poniżej znajduje się całe rozwiązanie:

function countVowels(text: string): number {

    let count = 0;
    for(let char of text.toLowerCase()) {
        count += Number("aeiou".includes(char));
    }
    return count;
}

Jasne, proszę pobrać!

Proszę zamienić wszystkie znaki spółgłoskowe na puste miejsca i obliczyć długość pozostałego ciągu. 

function countVowels(text: string): number {
    
    return text.replace(/[^aeiou]/gi, '').length;
}
